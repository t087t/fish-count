# Fish Count Study

このリポジトリは、釣れた魚の数に影響する要因を分析し、簡易的な予測モデルを構築する過程をまとめた学習用ノートブックである。

天気・気温・ユーザーごとの釣果データを用いて、探索的データ分析（EDA）、ポアソン回帰、クラスタリング、予測モデルの評価を学習した。

## 学習環境

- **Language**: Python
- **Environment**: Jupyter Notebook
- **Libraries**:
  - NumPy
  - pandas
  - Matplotlib
  - Seaborn
  - statsmodels
  - scikit-learn
- **Focus**: データ分析、可視化、回帰分析、クラスタリング、予測モデルの構築

## 分析の目的

本ノートブックでは、以下の2点を目的として分析を行った。

1. 天気や気温が釣れる魚の数に与える影響を調べる
2. 釣りがうまい人が晴れ・25℃の条件で何匹釣れるかを予測する

## 使用データ

天気、気温、ユーザー、釣れた魚の数に関する100件の観測データを使用した。

| カラム名 | 内容 |
|---|---|
| `fish_count` | 釣れた魚の数 |
| `weather` | 天気（`cloudy` または `sunny`） |
| `temperature` | 釣りをしたときの気温 |
| `user` | 釣りをしたユーザー（A～J） |

ユーザーは10人存在し、それぞれについて10回分のデータが記録されている。

### 2. 探索的データ分析（EDA）

#### 2.1 釣果の分布

ヒストグラムを用いて、釣れた魚の数の分布を可視化した。

釣果は0匹のケースが最も多く、釣果が増えるほど観測数が減少する右に歪んだ分布となっていた。

#### 2.2 気温と釣果の関係

折れ線グラフと散布図を用いて、気温と釣果の関係を確認した。

- 気温が高くなるほど釣果が増加する傾向がある
- 特に20℃以上では、高い釣果が観測されやすい
- 気温が高くなるほど、釣果のばらつきも大きくなる

#### 2.3 天気と釣果の関係

箱ひげ図と基本統計量を用いて、晴れの日と曇りの日の釣果を比較した。

- 曇りの日は晴れの日より釣果の中央値が高い
- 曇りの日の方が釣果のばらつきも大きい
- 平均釣果は曇りの日が2.92匹、晴れの日が2.04匹であった

#### 2.4 ユーザーと釣果の関係

棒グラフと箱ひげ図を用いて、ユーザーごとの釣果を比較した。

- ユーザーごとに釣果の傾向が大きく異なる
- ユーザーA、C、Gは比較的高い釣果を示す
- ユーザーD、Fは釣果が低い傾向にある
- 平均値だけでなく、中央値やばらつきも確認することが重要である

#### 2.5 ユーザーと気温の関係

ユーザーごとに釣りをした際の気温分布を可視化した。

ユーザーによって釣りをしている気温条件が異なるため、単純な平均釣果だけでは純粋な釣りのスキルを判断できない可能性があることを確認した。

## 予測モデルの構築

### 1. 訓練データとテストデータへの分割

`train_test_split()`を使用し、データを以下の割合で分割した。

- 訓練データ：70%
- テストデータ：30%

ユーザーごとのデータ数に偏りが生じないように、`user`を基準とした層化分割を行った。

### 2. カテゴリ変数の前処理

天気とユーザーをモデルに入力するため、`pd.get_dummies()`を使用してダミー変数に変換した。

### 3. ポアソン回帰モデル

目的変数である釣果は、0以上の整数で表されるカウントデータであるため、一般化線形モデルによるポアソン回帰を使用した。

最初のモデルでは、以下の変数から釣果を説明した。

- 気温
- 天気
- ユーザー

```python
fish_count ~ temperature + weather + user
````

### 4. KMeansによるスキル分類

単純な平均釣果だけでユーザーのスキルを判断すると、暖かい日など魚が釣れやすい条件の影響を受ける可能性がある。

そこで、ポアソン回帰から得られたユーザーごとの係数を、天気や気温の影響を調整した「釣りのうまさ」の指標として使用した。

この係数に対してKMeans法を適用し、ユーザーを以下の3つのスキルグループに分類した。

* 初心者（beginner）
* 中級者（intermediate）
* 上級者（advanced）

### 5. スキルグループを用いたモデルの再構築

ユーザーIDの代わりにスキルグループを説明変数として使用し、最終的なポアソン回帰モデルを構築した。

```python
fish_count ~ temperature + weather + skill_group
```

ユーザーをスキルグループにまとめることで、モデルを単純化し、未知データに対する汎化性能の向上を目指した。

### 6. 釣果の予測

最終モデルを使用し、以下の条件における釣果を予測した。

* スキル：上級者
* 天気：晴れ
* 気温：25℃

予測結果は、約5.83匹となった。

### 7. モデルの性能評価

モデルの性能は、訓練データとテストデータに対する平均二乗誤差（MSE）を用いて評価した。

| データ    |  MSE |
| ------ | ---: |
| 訓練データ  | 2.03 |
| テストデータ | 2.99 |

訓練データよりテストデータの誤差が大きいものの、未知のデータに対しても一定の予測性能を持つことを確認した。
